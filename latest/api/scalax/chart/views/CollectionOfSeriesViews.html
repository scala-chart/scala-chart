CollectionOfSeriesViews

trait CollectionOfSeriesViews extends CollectionOfTimeSeriesViews with CollectionOfTimePeriodValuesViews with CollectionOfXYSeriesViews

Contains implicit views that convert collections of the simpler *Series datasets to their *SeriesCollection counterparts so they can be used by chart factories.
